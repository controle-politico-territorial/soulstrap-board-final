# Soul Strap — Resumo estratégico do moodboard

## Ordem do board
1. Tipografia
2. Elementos
3. Fotografia
4. Embalagens

## Leitura estratégica
- começa pela linguagem
- passa para o sistema visual
- expande para o universo imagético
- fecha na materialização física da marca

## Direção-base de tipografia
- principal: Aeonik / Söhne / Suisse Int’l
- apoio: Neue Haas Grotesk / Founders Grotesk / GT America
- evitar: decorativas, fashion vazias, futuristas demais, genéricas demais

## Norte de percepção
- precisão
- sofisticação discreta
- utilidade refinada
- contemporaneidade
